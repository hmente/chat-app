# chat-app
REST + Sockets-based chat service built on top of NodeJS/ExpressJS using MongoDB.

## 1. System Manual

### 1.1 Requirements
* NodeJS 
* NPM
* MongoDB

### 1.2 Run the project
1. Run the following command to start the backend server: ```npm start```
2. Access the backend server on http://localhost:3000

## 2. Run Tests
```
npm test
```
